# QuickBite SPA - Food Delivery App

## Project Overview
A Data-Driven Single Page Application built with Vue 3, TypeScript, and Tailwind CSS using the DummyJSON API.

## Tech Stack
* Vue 3 (Composition API)
* TypeScript
* Tailwind CSS
* Vite

## Installation & Setup
1. Clone the repository.
2. Run `npm install` to install dependencies.
3. Run `npm run dev` to start the local development server.
